## [0.1.0] - 2024-03-04

### 新增
- HTTP 请求测试功能（GET、POST、PUT、DELETE、PATCH 等）
- 集合和文件夹管理
- 环境变量支持
- 请求历史记录
- 自动化测试（状态码、JSON 字段、全局变量）
- 暗色模式支持
- cURL 命令导入
- 集合导入导出功能
- 代码编辑器（支持 JSON、XML、HTML）
- 响应数据查看（Pretty、Raw、Headers）

### 技术栈
- Vue 3 + Vite
- Tauri 2
- PrimeVue + TailwindCSS
- Pinia 状态管理
- CodeMirror 6 编辑器
